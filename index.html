Introdução ao HTML

O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas da web. Ele estrutura o conteúdo na web usando uma série de elementos e tags.

Principais Tags de Formatação

A tag b é usada para deixar o texto em negrito. Exemplo: Este texto está em negrito usando a tag b.

A tag strong é usada para deixar o texto em negrito e dar ênfase. Exemplo: Este texto está em negrito usando a tag strong.

A tag i é usada para deixar o texto em itálico. Exemplo: Este texto está em itálico usando a tag i.

A tag em é usada para deixar o texto em itálico e dar ênfase. Exemplo: Este texto está em itálico usando a tag em.

A tag mark é usada para destacar partes do texto. Exemplo: Este texto está destacado.

A tag small é usada para deixar o texto menor. Exemplo: Este texto está menor.

A tag del é usada para marcar o texto como excluído. Exemplo: Este texto está excluído.

A tag sub é usada para deixar o texto como subscrito. Exemplo: Este texto está subscrito.

A tag sup é usada para deixar o texto como sobrescrito. Exemplo: Este texto está sobrescrito.
